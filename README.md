# Emotion Detection App  
This project is part of the IBM course *Developing AI Applications with Python and Flask*.  
It uses IBM Watson NLP to detect emotions in text, packaged and deployed with Flask.

## Features
- Detects five emotions (anger, disgust, fear, joy, sadness)
- Flask web interface with error handling
- Unit tested and PEP8 compliant (10/10 PyLint score)
